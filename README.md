# Портфолио Аким Камышов

Персональное портфолио (визитная карточка) Full Stack разработчика.

## 🌟 Особенности

- **Современный дизайн** - Темная тема с градиентами и анимациями
- **Адаптивность** - Полная поддержка мобильных устройств, планшетов и десктопов
- **SEO оптимизация** - Метатеги, семантическая разметка
- **Быстрая загрузка** - Оптимизация изображений и код сплиттинг
- **Интерактивность** - Анимации с Framer Motion

## 🚀 Технологии

- **React 19** - Библиотека для создания пользовательских интерфейсов
- **TypeScript** - Надмножество JavaScript с типизацией
- **Tailwind CSS** - Utility-first CSS фреймворк
- **React Router** - Роутинг для Single Page Applications
- **Framer Motion** - Библиотека для анимаций

## 📁 Структура проекта

```
src/
├── pages/
│   ├── Home.tsx       # Главная страница
│   ├── About.tsx      # Страница "Обо мне"
│   ├── Projects.tsx   # Страница "Проекты"
│   └── Contact.tsx    # Страница "Контакты"
├── App.tsx            # Основной компонент с маршрутизацией
├── main.tsx           # Точка входа
├── index.css          # Стили Tailwind CSS
└── utils/
    └── cn.ts          # Утилиты для классов
```

## 🛠️ Установка и запуск

### Требования
- Node.js 18+
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Сборка для продакшена
```bash
npm run build
```

### Предпросмотр сборки
```bash
npm run preview
```

## 🚀 Деплой

Для деплоя на GitHub Pages:

1. Установите gh-pages:
```bash
npm install -D gh-pages
```

2. Добавьте в package.json:
```json
"homepage": "https://ваш-юзернейм.github.io/портфолио",
"scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d dist"
}
```

3. Сделайте деплой:
```bash
npm run deploy
```
